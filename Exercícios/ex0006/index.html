Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito ("b") ela não e semântica.

Nesta frase, tenho uma frase semântica em negrito usando a tag ("strong").

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag (não semãntica)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto Marcado

Podemos marcar um texto com a tag ("mark") para não utilizar style no "P", podemos usar no CSS. Ou continuar normamente usando a tag amarela

Texo Grande e pequeno

Para textos grande Grande, big não e mais usado!

Para textos pequenos usamos Pequeno ("small").

Texto deletado

podemos utilizar a tag ("div") para que o leitor desconsidere o texto excluido
