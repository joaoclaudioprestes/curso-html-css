Outras formatações

Código-Fonte

Para inserir code podemos utilizar o envelopar com a tag ("code") Strong.

Para utiliar o envelopar Crt+ Shift + P.

Para deixar a formatação original utilizar antes de ("code") a tag ("pre").

Como criar uma citação

Para criar uma situação utilizamos a tag ("q").
Ex:Creio no riso e nas lágrimas como antídotos contra o ódio e o terror. Charli Chaplin.

Para criar blocos de citações utilizar ("blokquote"). A seguir terá um exemplo:

Não creio, no sentido filosófico do termo, na liberdade do homem. Todos agem não apenas sob um constrangimento exterior mas também de acordo com uma necessidade interior. Albert Einstein

Abreviações

Utilizado para dar significado para as abreviações com a tag ("abbr").
